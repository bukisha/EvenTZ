# EvenTZ
Simple application for event promotion

Idea of app is to help users promote events that range from small birthday parties to large scale events/manifestations like concerts or sport events.Second goal of app is to help me try out new stuff and learn.

App is in a stage of development at has only basic functionalities,but i hope that in the future it will be upgraded with more features and that the codew ill be more refined.It is also in plan to add usage of various libraries like dagger or maybe even RxJava sometime in the future.
